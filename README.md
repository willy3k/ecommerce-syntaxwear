# Syntaxwear - Tênis e Sneakers Online 👟

![Logo Syntaxwear](./img/logo/Logo.svg)

Bem-vindo ao repositório da **Syntaxwear**, uma loja online (e-commerce) moderna e estilosa focada na venda de calçados, tênis e sneakers. Este projeto foi desenvolvido para oferecer uma experiência de compra visualmente atraente e funcional, com foco em design limpo e organização de código.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as tecnologias fundamentais da web, com foco em boas práticas de estruturação:

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização avançada utilizando:
  - **Variáveis CSS**: Para facilitar a manutenção de cores e fontes.
  - **CSS Flexbox e Grid**: Para criação de layouts responsivos e alinhamentos precisos.
  - **Metodologia de Componentes**: Organização dos estilos por blocos (header, footer, hero, etc.).
- **Google Fonts**: Utilização das fontes *Ubuntu*, *Roboto* e *Inter* para uma tipografia moderna.
- **Ícones em SVG**: Para garantir nitidez em qualquer resolução.

---

## ✨ Funcionalidades

- **Design Responsivo**: Menu hamburguer para dispositivos móveis e layout que se adapta a diferentes tamanhos de tela.
- **Seção Hero**: Banner principal de destaque para lançamentos (como o *Krypton One*).
- **Categorias Interativas**: Navegação rápida entre os estilos Casual, Esporte, Moderno e Futurista.
- **Grade de Produtos**: Exibição organizada de modelos em destaque.
- **Newsletter**: Área para captura de e-mails de clientes interessados em novidades.
- **Redes Sociais**: Links integrados para Instagram, WhatsApp, TikTok e Facebook.

---

## 📁 Estrutura de Pastas

A organização do projeto segue uma estrutura modular para facilitar o desenvolvimento:

```text
ecommerce-syntaxwear/
├── index.html          # Página principal do site
├── css/                # Pasta de estilos
│   ├── base.css        # Estilos base (corpo, tipografia geral)
│   ├── reset.css       # Reset para padronização entre navegadores
│   ├── variables.css   # Definição de cores e fontes globais
│   └── components/     # Estilos específicos por seção (Header, Footer, etc)
├── img/                # Ativos visuais
│   ├── banners/        # Imagens de fundo e destaque
│   ├── icons/          # Ícones de interface (sacola, usuário, redes sociais)
│   ├── logo/           # Logotipo oficial
│   └── products/       # Fotos dos produtos e categorias
└── README.md           # Documentação do projeto
```

## 🛠️ Como Executar o Projeto

Como este é um projeto de front-end estático (HTML e CSS), você pode visualizá-lo de forma muito simples:

1. Faça o download ou clone este repositório.
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` em qualquer navegador (Chrome, Edge, Firefox).

> **Dica**: Se estiver usando o VS Code, recomendo a extensão **Live Server** para ver as alterações em tempo real enquanto edita o código.
